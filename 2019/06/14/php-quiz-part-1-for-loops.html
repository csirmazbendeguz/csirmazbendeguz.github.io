Comma operator

In C, the comma operator is an operator that evaluates two expressions, and returns the result of the second one. For example:

This ancient operator did not stand the test of time. Not many programming languages support it.

PHP does not have a comma operator either... except in for loop conditions. For some strange reason, in for loop conditions the comma operator is correctly evaluated. Weird!

Thus, the expression $i < 3, $j < 2 is equivalent to $j < 2 ($i < 3 is evaluated but disregarded). The loop will only iterate twice.
